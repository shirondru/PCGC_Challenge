Enformer is not included in this repository because it is prohibitively large. You can download it using, at the time this was written, this link (https://tfhub.dev/deepmind/enformer/1) or go to the author's github (https://github.com/boxiangliu/enformer). 

Code provided in this repository for using Enformer loads Enformer from: https://tfhub.dev/deepmind/enformer/1
If working on a computer/server with access to the internet, then there is no need to download Enformer ahead of time.
